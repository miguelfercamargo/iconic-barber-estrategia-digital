# Presentación de estrategia digital - ICONIC BARBER

Página estática independiente para presentar la propuesta digital al dueño de ICONIC BARBER.

## Cómo abrir

Abrir `index.html` directamente en el navegador. No depende de Astro, build, servidor local ni dependencias.
La imagen `hero-bg.webp` está copiada dentro de esta carpeta para que la presentación sea autocontenida.

## Propósito

- Explicar por qué `iconicbarberbogota.com` debe ser el enlace central del negocio.
- Mostrar cómo se conectan Google Business Profile, Maps, Instagram, Facebook, WhatsApp y Beunik.
- Traducir SEO técnico, Open Graph, JSON-LD, Astro, seguridad, rendimiento y SEO para IA a beneficios comerciales simples.
- Presentar el acuerdo: desarrollo sin cobro, dominio comprado por ICONIC BARBER, backlink discreto a `miguelcamargo.dev`.
